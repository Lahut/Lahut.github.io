SPECIFICITY

CSS Specificity เป็นคอนเซปต์การเลือก CSS Selector ที่มี แต้ม เยอะที่สุด มาใช้กับ Element บนหน้าเว็บไซต์
ภาษา CSS เหมือนภาษาการเขียนโปรแกรมอื่น ๆ ที่อ่านโค้ดจาก บน – ล่าง แปลว่า ถ้าเขียน CSS Selector ซ้ำกัน 2 อัน
อันล่างจะถูกนำไปใช้งานเพราะใหม่กว่า แต่ความพิเศษของ CSS คือ Selector ที่แตกต่างกันแต่ละ Selector จะมี แต้ม ไม่เหมือนกัน
และเมื่อไหร่ที่ แต้ม ต่างกัน ลำดับการเขียนโค้ดจะไม่มีผลทันที

Example

HTML
<hr class="line" id="redline">

CSS
.line { color : black} >>>>>>ทำให้เส้นเป็นสีดำ


#redline { color : red} >>>>>>ทำให้เส้นเป็นสีแดง

.line {} มีแต้ม 10 แต้ม {}
#redline {} มีแต้ม 100 แต้ม
เพราะฉะนั้น CSS Property ใน #redline {} ที่มีแต้ม 100 จะถูกดึงมาแสดงผลทับของ .line {}(10แต้ม) ที่มีแต้มน้อยกว่า เพราะมีแต้มมากกว่านั้นเอง
